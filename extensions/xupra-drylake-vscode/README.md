# Xupra DryLake VS Code Extension

Editor client for Xupra DryLake.

Current scaffold includes:

- backend connection bootstrap
- project and job views
- workspace scan/import commands
- compatibility, export, pull, and deploy commands
- generated export writeback into the local workspace
- `npm run check:cursor` compatibility guardrail

This extension is designed to work with the Xupra DryLake backend and to remain compatible with Cursor through standard VS Code extension APIs.
